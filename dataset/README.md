
# データセットについて

このプロジェクトで使用している画像データセットは、  
Google検索で収集した画像に対して、以下のような前処理を施したものです：

- 回転、反転などの拡張処理
- 明るさ・コントラストの調整
- 背景の削除

これらの画像には著作権が存在する可能性があるため、  
一般公開は行わず、**学科アカウントを持つ利用者のみに限定して**、Google共有ドライブ上で管理しています。
